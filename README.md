# CampusManagementSystem

### 项目介绍
GIS综合实验开发项目
利用OpenstreetMap提供的公共数据接口，实现浏览、编辑、下载全国任意可查询校园的OSM数据

#### 软件架构
使用Javascript、HTML、CSS进行网站开发，使用Vue渐进式框架实现页面交互，采用leaflet实现地图浏览与展示，利用axios网络请求库和OSM数据接口，实现OSM数据的下载


#### 安装教程

1.  下载本仓库
2.  打开cmd,切换到该仓库目录下
3.  执行npm install
